# Probability Terminology

1. 樣本空間 (Sample Space)
    - 一個實驗的所有可能結果所構成的集合
    - 用 `S` 表示
    - E.g. 擲一枚硬幣，樣本空間為 {正, 反}
2. 事件 (Event)
    - 樣本空間的子集合
    - E.g. 擲一個骰子，令事件 A 為出現偶數點，A = {2, 4, 6}
    - 我們感興趣的部分
    - Simple Event: 一個元素的事件 (atomic event)
    - Compound Event: 由兩個以上的元素組成的事件
3. 機率 (Probability) => P(A) = #(Event) / #(Sample Space)
