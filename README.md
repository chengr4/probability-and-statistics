# Probability

## Three Axios of Probability 機率三公理

1. For any event A, P(A) >= 0
2. P(S) = 1 (S = 樣本空間)
3. if events A1, A2, ... 互斥 (互斥：不可能同時發生) => P(A1⋃A2⋃A3...) = P(A1)+P(A2)+P(A3)...

### Derived from three axios

- P(∅) = 0: Empty set

  加以 S \cup ∅ = S
  => P(S) = P(S \cup ∅) = P(S) + P(∅)
  => P(∅) = 0
